Sentinema Documentation
=======================

Welcome to the official documentation for **Sentinema**!

Sentinema is an open-source project that connects quantitative movie ratings
with the qualitative emotional reactions of audiences. It allows you to visualize
*why* viewers liked or disliked a movie through sentiment analysis and word clouds.
