Configuration Guide
===================

Configuration Options
---------------------

- Data path: default ``./data/``
- Outputs path: default ``./outputs/``
- Model configuration: (planned)
  - sentiment model type
  - keyword extraction method
  - language settings

How to Customize
----------------

- Edit a configuration file (e.g. ``config.yaml``) (planned)
- Or pass CLI arguments to ``main.py`` (planned)
