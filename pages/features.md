# ✨ Sentinema Feature Showcase

Sentinema는 감정 분석과 영화 추천 기능을 결합한  
오픈소스 기반 지능형 추천 시스템입니다.

---

## 🎭 1. 감정 분석 (Emotion Classification)
- 사용자가 입력한 텍스트를 기반으로 감정을 분류
- Positive / Negative / Neutral 등 기본 감정 제공
- ML/DL 기반 모델 사용 (향후 커스텀 모델 적용 가능)

---

## 🎬 2. 영화 추천 기능 (Movie Recommendation)
- 감정 분석 결과에 맞는 영화 추천
- 긍정/부정 감정에 따른 장르 기반 추천
- 확장 가능하도록 설계된 추천 로직

---

## 📦 3. 오픈소스 기반 구성
- GitHub에서 코드 및 리소스 모두 공개
- 누구나 Issue, PR, Discussion으로 기여 가능

---

## 🌐 4. 웹 기반 서비스 구조
- GitHub Pages + Jekyll 기반 홈페이지
- 사용자/기여자들이 쉽고 빠르게 프로젝트 이해 가능

---

## 🛠 5. 개발자 친화적인 구조
- 명확한 폴더 구조
- 문서 기반 ReadTheDocs 지원 예정
- 새 기여자가 쉽게 참여할 수 있는 구조

---

각 기능은 지속적으로 업데이트되고 있으며  
자세한 기술적 설명은 추후 문서(ReadTheDocs)에서 제공될 예정입니다.
