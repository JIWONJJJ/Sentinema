# IMDb Dataset Notice

This folder is intentionally left empty.

The IMDb Movie Reviews dataset is NOT included in this repository due to licensing restrictions.
To run the project, please download the dataset manually from Kaggle and place the files in this folder.

Kaggle link:
https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews

Expected files after download (example):
- imdb_reviews.csv  (or original CSV name from Kaggle)
- imdb_labels.csv   (if separated)
- train/test split files if generated by users

Do NOT upload the dataset to this repository. Redistribution of the IMDb dataset is not allowed.
