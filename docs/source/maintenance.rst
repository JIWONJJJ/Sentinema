Maintenance and Troubleshooting
===============================

Common Issues
-------------

- **Missing dataset**:
  Make sure the IMDb dataset is downloaded from Kaggle
  and placed under ``./data/``.

- **Package version conflicts**:
  Recreate the virtual environment and reinstall using
  ``pip install -r requirements.txt``.
